# Yang Academic Profiles — Dual Academic Website

This is a formal static academic website for two researchers. It can be hosted on GitHub Pages, Netlify, Vercel, or any standard web server.

## Files

- `index.html` — homepage
- `people.html` — dual academic profiles
- `research.html` — research programs
- `publications.html` — representative publications with search/filter
- `contact.html` — contact and academic links
- `assets/style.css` — website styling
- `assets/script.js` — navigation highlighting and publication search/filter

## How to edit quickly

1. Replace `Academic Partner` with the partner's full name.
2. Replace initials `AP` with the partner's initials.
3. Add the partner's title, institution, research fields, selected publications, and contact links.
4. Replace placeholder publication entries with exact citations and DOI links.
5. Add photos if desired. A simple way is to place images in `assets/` and replace the avatar `<div>` with:

```html
<img class="avatar" src="assets/your-photo.jpg" alt="Wenjing Yang portrait" />
```

## GitHub Pages deployment

1. Create a GitHub repository, for example `academic-profiles`.
2. Upload all files in this folder.
3. Go to Settings → Pages.
4. Select branch `main` and root folder `/`.
5. Save. The website will be published as a GitHub Pages URL.
